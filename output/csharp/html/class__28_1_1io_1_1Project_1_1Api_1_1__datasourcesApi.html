Creates a new data source

This method creates a new data source. According to the specified data source category, the connection to the data source will be tested within this method. If the test does not pass the credentials will not be stored and the reponse status code will be set to 422.

Currently, the default MongoDB data source cannot be created using this method.

Parameters
categoryThe data source category.
nameThe name of the data source. The data source name can contain any alphabetic letter, numbers, dots, or dashes, and must start with an alphabetic letter.
tokenA project token.
_defaultWhether the new data source will be the default one for its category. The default value is false.
bodyThe data sources credentials as JSON.
Returns

Updates a data source

This method updates a data source changing its name, whether it is default or not, or its credentials. At least one change, that is, one of the optional parameters, must be specified.

Currently, the default MongoDB data source cannot be modified through this method.

Parameters
categoryThe data source category.
datasourceThe data source name.
tokenA project token.
nameThe new name of the data source. If not specified the data source is not renamed.
_defaultWhether the data source should become (if true) or cease to be (if false) the default one for its category. If not specified the data source does not change its default status.
bodyThe new data sources credentials as JSON. If not specified the data sources credentials are not changed
